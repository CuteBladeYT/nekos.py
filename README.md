# nekos.py
A Python module that uses Nekos API

## Requirements
- Python 3.6 and up - https://www.python.org/downloads/

# Install
```
pip install nekos.py
```

# Usage example
```py
import nekos
print(nekos.get_neko("meow"))

>>> 'https://cdn.nekos.life/meow/02D1A.jpg'
```
